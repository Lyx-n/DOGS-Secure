# DOGS-Secure
Fork of DOGS that just adds a couple of extra features
### Features:
- Protection from editing the ModStorage to remove DOGS lock on next relog
### Things I might add:
- Even more protection from switching between DOGS versions to go around the anti cheat (Though that is really immoral, so I will have a big consent or not option for that) (Maybe like fingerprinting or even worse)
